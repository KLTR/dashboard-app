DashBoard API

Get Report By Date:

This is a GET route. find report by the date

key: date
format: dd-mm-yy
values: string
Example:
https://localhost:3000/getReportByDate/August 28th, 2017

Enter new report

This is a POST route. Enter a new report

key: {report data}
values: number
Example: 
https://localhost:3000/enterNewReport

Get All Reports:

This is a GET route. Gets you all the reports so far

No Values!
Example:
https://localhost:3000/getAllReports

Get Last Report:

This is a GET route. Gets the last report created

No Values!
Example: 
https://localhost:3000/getLastReport

Get Trending Findings Source:

This is a GET route. Gets trending from source with date

No Values!
Example:
https://localhost:3000/getFindingsBySource

Get Trending Findings Type:

This is a GET route. Gets trending by type with date

No Values!
Example:
https://localhost:3000/getFindingsByType
